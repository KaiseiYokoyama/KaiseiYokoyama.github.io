JoyFlick 操作方法

子音選択

コントローラのRスティックを操作し（倒す，あるいは押し込む），子音を選択します．

母音選択

Rスティックを操作したまま，Lスティックを操作し，母音を選択します．

文字入力

Lスティックから指を離し，文字を入力します．

特殊入力

ZRもしくはZLボタンを押して，最後に入力した文字を濁音->半濁音->小音->清音の順に変換することができます．

また，Bボタンを押して，最後に入力した文字を削除することができます．

アプリケーション使用方法

コントローラの接続

コントローラのシンクロボタン（ボタンの位置はこちら）を押し，コンピュータにコントローラを接続してください．

Windows

Windowsと本アプリケーションの両方がコントローラのモードを設定しようとするため，一手間かける必要があります．

より具体的には，コントローラの接続が済み，デバイスのセットアップのバルーンが表示されてから，デバイスの準備ができましたと表示されるまで，本アプリケーションの起動を控えてください．

アプリケーションの実行

ダウンロードした.zipファイルを解凍し，実行ファイルをダブルクリックしてください．

macOS

開発元不明のアプリケーションとして実行がブロックされることがあります．その場合，「システム環境設定」 -> 「セキュリティとプライバシー」 -> 「一般」からアプリケーションを実行してください．

Windows

発行元不明のアプリケーションとして実行がブロックされることがあります．その場合，「詳細情報」からアプリケーションを実行してください．

コントローラの認識

アプリケーションがコントローラを認識していることを確認してください．

測定開始

コントローラと心の準備ができたら，測定を開始してください．

リトライ

測定が終了した後，もう一度測定に挑戦することができます．

リトライでは，画面表示オンのJoyFlickの測定に挑戦できます．

タッチタイプ版では，画面表示オフのJoyFlickの測定に挑戦できます．
